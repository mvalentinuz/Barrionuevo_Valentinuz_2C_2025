Proyecto Integrador - Tendedero Automatizado

General Description

El proyecto consiste en el desarrollo de un tendedero de ropa automatizado controlado por microcontrolador, capaz de gestionar de forma inteligente el momento de extender o resguardar la ropa según las condiciones ambientales. Mediante un sensor de lluvia, el sistema detecta la presencia de lluvia y, en caso de ser necesario, acciona un mecanismo motorizado (servomotor) que retrae la ropa hacia el interior para evitar que se moje. El sistema también cuenta con un módulo Bluetooth que envía notificaciones al teléfono móvil, informando al usuario sobre los eventos de guardado o despliegue.
